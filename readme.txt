An easily way to print social icons without javascript to share on more populars social networks.

== Description ==

This plugin print social icons to share every post on most populars social networks witout load any javascript.  Just a clean link that open a new window.  You must call the function on your template inside 'the loop'.

<?php if(function_exists('wpe_socialicons')) wpe_socialicons(); ?>

Supported Social Networks:

* Facebook.
* Twitter.
* Google+.
* LinkedIn.
* Pinterest.

== Installation ==

1. Download
2. Upload to your `/wp-contents/plugins/` directory.
3. Activate the plugin through the 'Plugins' menu in WordPress.
